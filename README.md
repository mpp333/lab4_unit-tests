# Лабораторная работа №4: Написание Unit-тестов

## Описание
Проект содержит реализацию функций для вычисления площади и периметра прямоугольника с юнит-тестами.

## Функции
- `area(a, b)` - вычисляет площадь прямоугольника
- `perimeter(a, b)` - вычисляет периметр прямоугольника

## Тестирование
Для тестирования используется модуль `unittest` Python.

### Запуск тестов
```bash
python -m unittest rectangle.py -v
